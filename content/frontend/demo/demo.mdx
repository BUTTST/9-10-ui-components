# 示例元件

這是一個用於測試系統的簡單示例。

## 基礎實作

```tsx
interface DemoProps {
  title: string
  children: React.ReactNode
}

export const Demo = ({ title, children }: DemoProps) => {
  return (
    <div className="bg-white p-4 rounded-lg shadow">
      <h2 className="text-lg font-semibold">{title}</h2>
      {children}
    </div>
  )
}
```

## 使用方式

```tsx
<Demo title="示例標題">
  <p>這是示例內容</p>
</Demo>
```
